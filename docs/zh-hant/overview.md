# 概述

TanStack Store 是一個框架無關 (framework agnostic) 的資料儲存庫，隨附針對主流框架如 React、Solid、Vue、Angular 和 Svelte 的專用適配器 (adapters)。

TanStack Store 主要用於大多數框架無關的 TanStack 函式庫內部狀態管理 (state management)。它也可以作為獨立函式庫用於任何框架或應用程式。
